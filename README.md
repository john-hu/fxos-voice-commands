# fxos-voice-commands

## Setup

```bash
$ git clone git@github.com:jedireza/fxos-voice-commands.git
$ cd fxos-voice-commands
$ npm install
```

## Build

```bash
$ npm run build
```

## Local dev server

To serve the `./build` directory locally run:

```bash
$ npm start
```

Now open http://localhost:8080/ in your navigator of choice.
